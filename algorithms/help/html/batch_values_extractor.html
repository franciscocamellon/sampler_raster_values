This algorithm processes vector point data against raster datasets, extracting pixel values and storing them in a specified field. It also provides the option to create a new field for the variable of interest.

• Images Folder: Choose the folder containing raster datasets for analysis. • Input Point Layer: Select the vector point layer representing observation locations. • Field with the Observation Date: Specify the field in the point layer that contains observation dates. • Field to Save the Pixel Value: Specify the field in the point layer to store the pixel values from the raster. • Create a Field for Variable of Interest: Optionally create a new field for the variable of interest (default name: 'SAMPLE'). • AQUA MODIS Variable: Choose the AQUA MODIS variable for extracting pixel values.
