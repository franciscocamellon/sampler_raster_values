This algorithm processes vector point data against a single raster layer, extracting pixel values and storing them in a specified field. It provides the option to create a new field for the variable of interest. The "Process Only Selected Features" option allows users to focus on specific features if desired.

• Input Raster: Select the raster layer containing the data for analysis. • Input Point Layer: Select the vector point layer representing observation locations. • Process Only Selected Features: Toggle to process only the selected features in the input point layer. • Field for Variable of Interest: Specify the field in the point layer that contains the variable of interest. • Create a Field for Variable of Interest: Optionally create a new field for the variable of interest (default name: 'SAMPLE').
