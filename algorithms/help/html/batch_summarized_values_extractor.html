This algorithm is designed to compute summarized statistical values from raster data associated with point locations.

• Points Spreadsheet: Select the spreadsheet containing point data. Choose a file with the necessary point information. • Date Field: Specify the field in the spreadsheet that contains date information. • Images Folder: Choose the folder containing images related to the point data. Select the folder where your images are stored. • Raster Band: Select the raster band for analysis. Choose from available options: '1', '2', '3'. • AQUA MODIS Variable: Choose the AQUA MODIS variable for analysis. Select the variable of interest from the options provided. • Output Folder: Specify the folder where the processed output will be saved.
